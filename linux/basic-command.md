```
command [options] [arguments]
command는 명령어
options는 명령어 옵션
arguments는 명령어에 전달되는 인자
```
- ls 디렉토리 내용 보여주기
    - -l 파일 상세 정보
    - -a 숨김 파일 표시시
- cd 디렉토리 변경
    - 디렉토리 명
    - .. 상위 폴더
    - 알파벳 누르고 tab 누르면 자동 완성 (단, 중복되면 중복되는 단어까지만 자동완성)
- mkdir 현재 디렉토리에 새로운 디렉토리 생성
- rm 파일 또는 디렉토리 지우기
    - -r 디렉토리 지우기
- pwd 현재 디렉토리 전체 경로를 출력
- touch 파일 생성
- cat 파일 내용 보여주기
